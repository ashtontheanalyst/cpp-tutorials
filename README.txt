Development Setup:

   - Make sure g++ is downloaded (our compiler)
   - Ready to run

Making a File and Running It:
   - In the local directory
   - mkdir folderName
   - cd folderName
   - touch fileName.cpp
   - Put the code in that file and save
   - g++ fileName.cpp
   - ./a.out
   - OUTPUT
